Maximum Submatrix Project Documents
===================================

This is the documentation directory for the Maximum Submatrix project. It
contains the following documents:

report.pdf - The final report for the project. This is the document that will be
             graded.

doxygen.pdf - Doxygen generated documentation for the project. This is not the
              final report. But when you have questions about the code, this is
              the first place to look.
