This function frees the memory allocated for the matrix structure and the matrix elements.

There is pointer in the matrix structure, so you need to use this function to free the memory allocated for the matrix.

\mbox{\hyperlink{structMatrix}{Matrix}} pointer should be set to NULL if not been initialized or was freed.

This function will check if the pointer is NULL and prevent double free, so you need not to check it before calling this function. \mbox{\Hypertarget{mss_8h_a09b5ab66a616de9b29b1631ee4eb0c67}\label{mss_8h_a09b5ab66a616de9b29b1631ee4eb0c67}}
