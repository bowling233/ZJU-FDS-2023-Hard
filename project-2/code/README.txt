AutoGrad Project Source Code 
=====================================

This is the source code for the AutoGrad Project. The project is written in C
and uses the GNU Make build system. Here is a list of the source files:

main.c - The driver program.

expression.h - The header file for the main algorithms and data structures.

expression.c - The implementation of functions in the header.

Makefile - The GNU Make build system file. It contains the rules for
           building the project.

In order to build the project, simply run the following command in UNIX systems:

    make
